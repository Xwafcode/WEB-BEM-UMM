# Web BEM UMM - Kabinet Satya Prabaswara

Situs web resmi Badan Eksekutif Mahasiswa Universitas Muhammadiyah Malang (BEM UMM) periode 2024-2025, Kabinet Satya Prabaswara. Portal utama untuk seluruh informasi, berita, kegiatan, dan profil kepengurusan gerakan mahasiswa UMM.

## 🚀 Fitur Utama
- **Beranda Interaktif**: Animasi *parallax* modern, sorotan berita, dan profil pimpinan BEM.
- **Berita & Artikel**: Sistem filter dan kategorisasi artikel terbaru dari BEM.
- **Struktur Organisasi**: Bagan kepengurusan hierarkis yang interaktif untuk desktop dan mobile.
- **Responsif & Estetik**: UI premium, transisi halus, didukung animasi ScrollReveal berbasis *framer-motion*.

## 🛠️ Teknologi yang Digunakan
- **[React](https://react.dev/)** + **[Vite](https://vitejs.dev/)** - Framework utama & build tool.
- **[Tailwind CSS](https://tailwindcss.com/)** - Styling framework.
- **[Framer Motion](https://www.framer.com/motion/)** - Animasi *scroll*, *parallax*, dan *page transition*.
- **[React Router](https://reactrouter.com/)** - Manajemen navigasi (Routing).
- **[Phosphor Icons](https://phosphoricons.com/)** - Koleksi ikon.

## 📦 Instalasi & Menjalankan Lokal

1. **Clone repository ini**
   ```bash
   git clone https://github.com/Xwafcode/WEB-BEM-UMM.git
   cd WEB-BEM-UMM
   ```

2. **Install dependency**
   ```bash
   npm install
   ```

3. **Jalankan *Development Server***
   ```bash
   npm run dev
   ```

4. Buka `http://localhost:5173` di browser Anda.
